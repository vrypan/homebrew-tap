# homebrew-tap

A Homebrew tap for installing custom formulae.

## Installation

```bash
brew tap vrypan/tap
```

## Available Formulae

### Bckt

**Version:** 0.6.4

**Description:** bckt is an opinionated but flexible static site generator for blogs

**Project page:** https://github.com/vrypan/bckt

**License:** MIT

**Installation:**
```bash
brew install vrypan/tap/bckt
```
